PROJECT TITLE: Laboratory Exercise 6
PURPOSE OF PROJECT: Merge sort and quick sort comparison for Worker class and Contact class
VERSION or DATE: October 17, 2019
HOW TO START THIS PROJECT: Run in BlueJ or terminal with input " java ExperimentController fileName averageNum numberOfItems0 numberOfItems1 ... numberOfItemsN" for terminal.
                                                                {"fileName", "averageNum", "numberOfItems0", "numberOfItems1", ... ,"numberOfItemsN"}
                           fileName is a String input that indicates the filename the data has to be written into
                           averageNum is the number of times the experiment is to loop to collect data and average the sum of data collected by that value.
                           numberOfItems0 to numberOfItemsN is number of items to be sorted.
