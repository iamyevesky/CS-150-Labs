PROJECT TITLE: Lab 12. Hash Table Implementation
PURPOSE OF PROJECT: Testing the performance of insert and search for different has table implementation
VERSION or DATE: December 8, 2019
HOW TO START THIS PROJECT: Run from terminal BlueJ the main function with parameters "java ExperimentController data.csv input.txt output"
                           where data.csv is the array containing names and id for ExperimentController
                           input.txt is the input file with format: initial size of hash table, load factor for LinearHashTable, maxChainValue of ChainHashTable
                           output is the name of the output file in .csv format dubbed: "output.csv"
                           Input file data is of format
                           10 0.3 3
                           10 0.5 4
                           10 0.75 5
